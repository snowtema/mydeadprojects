# Signature

Interface and boundary patterns: signature stack, boundary norms, slot discipline, epistemic morphisms.

## When to Load This Domain

**Load signature when you need to:**
- Design interfaces and boundaries
- Handle contracts and service clauses
- Manage slot/value/reference discipline
- Work with epistemic morphisms and views
